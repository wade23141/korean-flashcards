# 韓文單字測驗機 (Korean Flashcard Quiz)

A simple front-end flashcard quiz for learning Korean vocabulary. Includes:
- index.html (UI)
- style.css (responsive styles + flip animation)
- script.js (lesson data, flip/next/prev, shuffle, localStorage)

How to run
1. Clone the repo:
   git clone https://github.com/wade23141/korean-flashcards.git
2. Open `index.html` in a browser.

Features
- Lesson selection
- Flip card to reveal Chinese/Hanja
- Prev/Next navigation and keyboard support
- Optional shuffle and state persistence in localStorage

Customization
- Edit lessons in `script.js` or replace with an external JSON loader.
- Optional extensions: TTS (Web Speech API), CSV import/export, spaced-repetition.
